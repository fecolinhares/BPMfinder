# Quick Plan: SEO + Performance + Code Quality

## Goal
Apply SEO skill (addyosmani/web-quality-skills) and performance optimization (impeccable optimize reference) to BPMfinder.

## SEO Improvements
1. robots.txt — allow all crawlers
2. sitemap.xml — single page sitemap
3. <link rel="canonical"> in <head>
4. JSON-LD WebApplication structured data
5. <meta name="robots" content="index, follow">
6. aria-labels on icon-only elements

## Performance Improvements
1. <link rel="preconnect" href="https://cdn.jsdelivr.net">
2. <link rel="dns-prefetch" href="https://cdn.jsdelivr.net">
3. Scripts: async/defer on non-critical
